# TerraTogether

TerraTogether is a multiplayer mod for Alabaster Dawn. 

⚠️ **Work in Progress!** This mod is currently still in active development! 

You need [TerraModloader](https://github.com/Serverator/terra-modloader) for it to work. This is temporary, soon it will move to proper [Selene Modloader](https://github.com/Project-Selene/Project-Selene)!

## Features

### Currently synced
- World position
- Animations
- Characters and skins
- Effects (except for looped ones)

### Planned features
- Sound sync
- Player name display
