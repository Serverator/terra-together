# TerraTogether

TerraTogether is a multiplayer mod for Alabaster Dawn. 

You need [TerraModloader](https://github.com/Serverator/terra-modloader) for it to work.

⚠️ **Work in Progress!** This mod is currently still in active development! 

## Features

### Currently synced
- Player position
- Player animations
- Player skins

### Planned features
- Show other player weapon effects
- Play sounds from other players 
- Player name display
